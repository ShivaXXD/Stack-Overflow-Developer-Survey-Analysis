# Stack Overflow Developer Survey Analysis

## Project Overview
This project involves an in-depth analysis of the **Stack Overflow Global Developer Survey** dataset. The goal was to identify key trends in programming languages, income levels, and work formats to understand the modern developer landscape.

## Tools & Technologies
* **Python:** Core language for data manipulation and analysis.
* **Pandas:** Data cleaning, filtering, and aggregation.
* **Jupyter Notebook:** Environment for interactive data exploration and documentation.

## My Tasks
* **Data Processing & Visualization:** Cleaned large-scale survey data and created visual representations of key metrics.
* **User Behavior Analysis:** Analyzed how developers choose technologies and work environments.
* **Insights & Trends:** Identified correlations between tech stacks, industries, and compensation.

## Key Results
* **Python Popularity:** Found that **37.5% of respondents** use Python, highlighting its dominance in the industry.
* **Top-Paid Specialist Profile:** Identified that the highest-paid roles are typically **Remote** positions in **Software Development** or **Fintech**.
* **Education vs. Income:** Proved that high income levels are primarily driven by **technology stack and industry expertise**, rather than the presence of a formal scientific degree.
